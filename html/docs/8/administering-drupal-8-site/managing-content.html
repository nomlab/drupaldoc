Managing content

Adding content

このガイドではインストール時に Basic Page と Article の2つのコンテンツタイプを定義する standard profile を使用したことを前提としています．(もし，minimal profile を使用したならば，最初にコンテンツタイプを定義する必要があります．)

始める前に: コンテンツを作成する権限を持つユーザでログインしていることを確認してください．-もし分からなければ，システム管理者に問い合わせてください．そうしなければ，選択する必要があるいくつかの領域が表示されません．

  1. フロントページから Add content を選択する．(ナビケーションメニューもしくは管理者メニューから)
  1. これにより，追加するコンテンツのタイプを尋ねる画面が表示されます．デフォルトでは，Drupal 8 には Article と Basic Page の2種類のコンテンツタイプがあります．
  1. Article を選択します．Basic Page と Article の唯一の大きな違いは，Articleを作成するときには画像をアップロードでき、Basic Page は静的ページ用です．コンテンツタイプの領域は後から更新可能です．
  2. Articleの情報を入力するフォームが表示されます．リビジョンログ，メニュー，コメント，著者 および プロモーションオプションの設定も可能です．これらの設定は，Drupalインスタンスの拡張やコントリビュートされたモジュールの使用によって変化します．
  1. Title テキストボックスにページのタイトルを入力します．:例えば，Test Article.
  2. Summary 領域に記事を簡潔に表すテキストを入力します．
  3. Body 領域にテキストを入力します．もし既に書いたテキストがあるならば，この領域にカット＆ペーストするだけです．書式設定オプションは，テキスト領域の下にある Text Fromat のドロップダウンから選択できます．デフォルトで利用可能なオプションは，Basic HTML，Restricted HTML と Full HTML です．
  4. コンテンツにタグを付けることができます。 タグ付けについて知るにはここをクリックしてください。
  5. コンテンツタイプに画像をアップロードできます．
  6. Save をクリックします．Save ボタンからコンテンツを公開したり非公開のままにしたりすることができます．
  7. 自身の投稿を見るべきです．

Managing nodes

1つ以上のノードに様々な操作を実行できます．ノードを公開または非公開として指定したり，フロントページに昇格したり降格したり，最上部に固定したり(リストの先頭に固定)することができます．また，1つ以上のノードの削除もできます．

管理者メニューからコンテンツリンクをクリックするか '/admin/content' に移動することで，Drupal アプリケーション内の全てのコンテンツのリストが表示されます．

  • Administration > Content をクリックします．
  • 必要に応じて，表示されたコンテンツを以下の様にステータス、タイプ，タイトル，言語でフィルタリングします．
    • フィルタの any / all の内，目的の値を選択します．
    • フィルタリング後のコンテンツを表示するため Filter ボタンをクリックします．
  • 管理するコンテンツは各行の左側にあるボックスを使用して選択します．
  • 全てのコンテンツを選択するには，Title の左側にあるボックスをクリックします．
  • 更新オプション領域内のアクション選択ボックスから実行する管理更新オプションを選択します．例えば，:Remove content from front page．
  • Update をクリックします．

ノードのアクセス許可の情報については，Mnaging Users セクションを参照してください．

Managing comments

全てのコメントのリストビューを表示できます．リストから，1つ以上のコメントの非公開や削除を選択できます．

  • Administration > Content をクリックします．
  • Comments タブをクリックします．
  • 1つ以上のコメントを選択します．
  • ドロップダウンリストから非公開や削除を選択します．
  • Update をクリックします．
